# Findhat

A simple terminal game where you explore a field to find the hat (^) while avoiding holes (O).

## How to Play
- You are represented by the '*' character.
- Move using:
  - `w` = up
  - `s` = down
  - `a` = left
  - `d` = right
- The goal is to reach the hat (^) without falling into holes (O).

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the game:
   ```bash
   node main.js
   ```

## Features
- Randomly generated fields
- Adjustable field size and hole percentage
- Simple command-line controls

## Example
```
*░░O
░O░O
░░░O
O░░░
░░O░
░^░░
```
